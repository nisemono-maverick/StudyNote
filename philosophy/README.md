# 西方哲学史学习笔记

> 主教材：斯通普夫《西方哲学史》

## 学习记录方式

以 **哲学家 → 核心问题 → 思想/观点 → 推导过程 → 对话关系** 为主轴记录。每读完一个哲学家，在对应时期目录下新建笔记文件，以 `哲学家姓名.md` 命名。

## 目录结构

| 目录 | 内容范围 | 代表哲学家 |
|------|---------|-----------|
| `01-古希腊罗马哲学/` | 前苏格拉底 → 希腊化/罗马时期 | 泰勒斯、巴门尼德、柏拉图、亚里士多德、伊壁鸠鲁、斯多葛、怀疑论 |

| `04-德国古典哲学/` | 康德 → 黑格尔 | 康德、费希特、谢林、黑格尔 |
| `05-现代哲学/` | 马克思主义、功利主义、存在主义、分析哲学等 | 马克思、密尔、尼采、维特根斯坦、海德格尔等 |

## 使用流程

1. **读教材**：按斯通普夫的章节推进，读完一个哲学家后，复制 `../00-笔记模板.md` 的内容。
2. **新建笔记**：在对应时期目录下新建 `哲学家姓名.md`，粘贴模板并填空。
3. **标记状态**：笔记顶部标注 `status: 初稿 / 待补充 / 已整理`。
4. **定期复盘**：积累一定数量后，用 `../99-脉络梳理.md` 整理横向对比（如"柏拉图与亚里士多德的方法论分歧"）。

## 学习原则

- **不追求完美**：初稿允许只记关键词和疑问，后面再补推导过程。
- **多问"他在回应谁"**：哲学是对话史，孤立记一个哲学家容易失真。
- **标记困惑**：模板的「我的疑问」栏必须诚实填写——这是真正理解哲学的入口。
