## 第一阶段：概念奠基期（第1-2个月）

### 核心目标
建立拉康理论的「基本地图」，能准确解释：镜像阶段、想象界/象征界/实在界、欲望、他者、对象a、能指链。

### 学习节奏
- **理论学习**：每周3-4小时
- **视频辅助**：每周1-2小时
- **案例联结**：每周看1部相关影片或片段，做概念对应笔记

## 第三阶段：应用拓展期（第6个月起，持续）

### 核心目标
将理论系统应用于电影动漫分析，形成独立见解和个人风格。

### 阅读任务（根据兴趣选读）
- [ ] 齐泽克《斜目而视：透过通俗文化看拉康》
- [ ] 克里斯蒂安·麦茨《想象的能指》（电影理论方向）
- [ ] 斋藤环《战斗美少女的精神分析》（动漫方向）
- [ ] 保罗·范霍恩《孤独时代的爱》（爱情/欲望方向）

### 分析实践：五步分析法
对每部深入分析的作品，按以下步骤执行：
1. **前置理论准备**：明确概念，阅读导论 → 产出：概念笔记
2. **初次观影与标记**：自然观看，标记异常时刻 → 产出：场景标记清单
3. **理论-文本对照**：选2-3个核心场景细致分析 → 产出：分析草稿（500-800字）
4. **批判性反思**：评估阐释力与局限 → 产出：反思笔记
5. **输出与讨论**：撰写完整文章并分享 → 产出：成文（1500-3000字）

## 避坑检查表

- [ ] **不过早读原著**：第一阶段结束前，不碰《拉康选集》原文
- [ ] **不纯理论化**：定期读临床导向著作，记住拉康是实践理论
- [ ] **不生搬硬套**：分析电影时，先描述文本细节，再引入概念
- [ ] **不追求完全理解**：允许暂时不懂，把困惑记录下来
- [ ] **防止兴趣枯竭**：学累了就回到喜欢的电影/动漫，用新概念重新分析
