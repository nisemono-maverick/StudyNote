# 文本表示

> 核心问题：如何把非结构化的文本，转化为机器能处理的结构化向量？

---

## 3.1 概述
- 文本表示是将自然语言转化为计算机能够理解的数值形式
- 早期文本通常将整段文本编码为一个向量（如词袋模型），此类方法实现简单、计算高效，但是无法表达语序和小下文语义
- 文本第一步通常是分词和词表构建
	- **分词（Tokenization）** 是将原始文本切分为若干具有独立语义的最小单元（即token）的过程，是所有 NLP 任务的起点。
	- **词表（Vocabulary）** 是由语料库构建出的、包含模型可识别 token 的集合。词表中每个 token 都分配有唯一的 ID，并支持 token 与 ID 之间的双向映射。
![[Pasted image 20260604002113.png]]
- 后续模型构建过程中，模型会先对输入文本进行分词，再通过词表将其转换为词向量，最终生成结果。常常结果通过概率表示，系统选择最大出现概率，最后输出文本
![[Pasted image 20260604002130.png]]

## 3.2 分词
### 英文分词
#### 词级分词
- 按词语进行切分。空格、标点常常是天然的分割符
- OOV（Out-of-Vocabulary）：在模型推理阶段，输入文本中可能会出现不在预料库词表中的词
- 通常处理方式是将其统一替换为特殊标记，从而导致语义丢失
#### 字符级分词
- 以单个字符为最小单位进行分词
- 但是切成单个字符语义信息就会很弱了。输入序列也会很长
#### 子词级分词
- 将词语切成子词，如词根、前缀、后缀或常见词片段
- tiktokenizer.vercel.app 体验大模型分词算法
![[Pasted image 20260604225218.png]]
- 常见子词分词算法包括 BPE、WordPiece、Unigram Language Model
- BPE 的基本思想：
	- 在训练阶段，将语料拆成单个字符，构架初始词表；然后迭代的将出现频率高的相邻字符合并为子词单元加入词表，直到达到预设上限
	- 在分词阶段，先将语料拆成最小单位，然后根据训练所得规则逐步合并，直到无法继续
	- 详细介绍见[[BPE tokenization 算法]]

### 中文分词
#### 字符级分词
- 将文本按照单个汉字进行切分，每个汉字都视为独立的 token
- 由于汉字本身具有独立语义，所以字符级分词天然具有可行性
#### 词级分词
- 将中文文本按照完整词语进行切分，由于中文没有空格，词级分词依赖辞典、规则或模型来识别词语边界，与英文词级分词存在同样的问题
#### 子词级分词
- 同样可以套用类似 BPE 算法的思路对中文进行分词

- 在大模型出现之前，中文分词常常用分词器，进行词级分词

### 分词工具
- 基于辞典或模型的传统词级分词方法（jieba, HanLP）
- 基于子词建模算法的子词级分词方法 (HuggingFace Tokenizer、 SentencePiece、 tiktoken)

#### jieba 分词器
##### 分词模式
- 精确模式。试图将句子最精确地切开，适合文本分析。
- 只需导入 `jieba` 包，然后调用 `cut` 和 `lcut` 方法
```python
import jieba
text = "小明毕业于北京大学计算机系"
word_generator = jieba.cut(text) # cut方法返回一个生成器
for word in word_generator:
    print(word)
    
word_list = jieba.lcut(text) # lcut方法返回一个列表
print(word_list)
# ['小明', '毕业', '于', '北京大学', '计算机系']
```
- 全模式。把句子中所有的可以成词的词语都扫描出来
- 只需在 `cut` 和 `lcut` 方法中额外添加参数 `cut_all=True`
```python
word_list = jieba.lcut(text, cut_all=True)
print(word_list)
# ['小', '明', '毕业', '于', '北京', '北京大学', '大学', '计算', '计算机', '计算机系', '算机', '系']
```
- 搜索引擎模式。在精确模式基础上，对长词进一步切分，适合用于搜索引擎分词
- 调用 `cut_for_search` 方法
```python
word_list = jieba.lcut_for_search(text)
print(word_list)
# ['小明', '毕业', '于', '北京', '大学', '北京大学', '计算', '算机', '计算机', '计算机系']
```
##### 自定义词典
- jieba支持用户自定义词典，以便包含 jieba 词库里没有的词，用于增强特定领域词汇的识别能力
- 自定义词典格式：一个词占一行三列，三列分别是词语、词频、词性标签
