## 5.1 感知机（Perceptron）

### 什么是感知机？

**感知机（Perceptron）** 是神经网络的最小单元，由 Frank Rosenblatt 于 1957 年提出。它本质上是一个**线性二分类器**。

### 数学模型

$$
\hat{y} = \text{step}(w_1 x_1 + w_2 x_2 + ... + w_n x_n + b) = \text{step}(\mathbf{w} \cdot \mathbf{x} + b)
$$

其中：
- $\mathbf{x} = (x_1, x_2, ..., x_n)$：输入特征向量
- $\mathbf{w} = (w_1, w_2, ..., w_n)$：权重向量
- $b$：偏置（bias）
- $\text{step}(z)$：阶跃函数，$z \geq 0$ 时输出 1，否则输出 0

### 直观理解

感知机就是在特征空间中画一条**直线**（或超平面），把两类数据分开：

```
        ✕ ✕ ✕
    ✕ ✕   |   ✕
  ✕ ✕     |     ✕
----------+----------  ← 决策边界（w·x + b = 0）
      ○   |   ○ ○
        ○ | ○ ○ ○
          |○ ○
```

### 感知机的局限

| 问题 | 说明 |
|------|------|
| **只能处理线性可分数据** | 无法解决 XOR 问题 |
| **阶跃函数不可导** | 无法用梯度下降优化 |
| **没有隐藏层** | 无法学习复杂的特征组合 |

> **关键洞察**：感知机的局限催生了**多层神经网络**——用多个感知机堆叠，并引入平滑的激活函数，就能逼近任意复杂的非线性函数（万能近似定理）。

## 5.2 前馈神经网络（Feedforward Neural Network）

### 从感知机到多层网络

**前馈神经网络（FFNN）**，也叫**多层感知机（MLP）**，由三层组成：

```
输入层          隐藏层           输出层

 x₁ ──→    ┌───┐
           │ h₁│ ──┐
 x₂ ──→    ├───┤   │
           │ h₂│   ├──→  ┌───┐
 x₃ ──→    ├───┤   └──→  │ y │
           │ h₃│ ──┐     └───┘
 x₄ ──→    └───┘   │
                   └──→
```

- **输入层**：接收原始特征（如 TF-IDF 向量）
- **隐藏层**：学习特征的非线性组合（可以有多层）
- **输出层**：产生最终预测（如分类概率）

### 前向传播（Forward Propagation）

数据从输入层流向输出层的过程：

$$
\mathbf{h} = \sigma(\mathbf{W}^{[1]} \mathbf{x} + \mathbf{b}^{[1]})
$$

$$
\hat{y} = \sigma(\mathbf{W}^{[2]} \mathbf{h} + \mathbf{b}^{[2]})
$$

其中 $\sigma$ 是激活函数（见 5.3 节）。

### 代码实战：用 PyTorch 搭建简单 FFNN

```python
import torch
import torch.nn as nn

class SimpleFFNN(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super(SimpleFFNN, self).__init__()
        # 第一层：输入 → 隐藏层
        self.fc1 = nn.Linear(input_dim, hidden_dim)
        # 激活函数
        self.relu = nn.ReLU()
        # 第二层：隐藏层 → 输出层
        self.fc2 = nn.Linear(hidden_dim, output_dim)
        # 输出概率分布
        self.softmax = nn.Softmax(dim=1)
    
    def forward(self, x):
        h = self.relu(self.fc1(x))   # 隐藏层输出
        out = self.fc2(h)             # 输出层（原始分数）
        return self.softmax(out)      # 转成概率

# 示例：输入 100 维 TF-IDF 向量，隐藏层 50 维，输出 2 类（正面/负面）
model = SimpleFFNN(input_dim=100, hidden_dim=50, output_dim=2)

# 模拟一个输入（batch_size=1, input_dim=100）
x = torch.randn(1, 100)
output = model(x)
print(output)  # tensor([[0.48, 0.52]])，两类概率
```

### ⚠️ 注意：隐藏层的作用

| | 无隐藏层（感知机） | 有隐藏层（FFNN） |
|---|------------------|-----------------|
| **表达能力** | 线性决策边界 | 非线性决策边界 |
| **特征学习** | ❌ 无 | ✅ 自动学习特征组合 |
| **XOR 问题** | ❌ 无法解决 | ✅ 可以解决 |
| **参数量** | 少 | 多（容易过拟合） |

## 5.3 激活函数（Activation Functions）

### 为什么需要激活函数？

如果没有激活函数，多层网络等价于单层线性变换：

$$
\hat{y} = W_2(W_1 x + b_1) + b_2 = (W_2 W_1)x + (W_2 b_1 + b_2) = W'x + b'
$$

**激活函数引入非线性**，让网络可以拟合任意复杂的函数。

### 常见激活函数对比

| 激活函数 | 公式 | 图像特点 | 优点 | 缺点 |
|---------|------|---------|------|------|
| **Sigmoid** | $\sigma(x) = \frac{1}{1 + e^{-x}}$ | S 型曲线，输出 (0, 1) | 平滑可导，可解释概率 | 梯度消失，非零中心化 |
| **Tanh** | $\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$ | S 型曲线，输出 (-1, 1) | 零中心化，比 Sigmoid 好 | 仍有梯度消失问题 |
| **ReLU** | $\text{ReLU}(x) = \max(0, x)$ | 负半轴为 0，正半轴线性 | 计算快，缓解梯度消失 | 神经元可能"死亡" |
| **Leaky ReLU** | $\max(\alpha x, x)$ | 负半轴有小斜率 | 解决 ReLU 死亡问题 | 需要调 $\alpha$ 参数 |

### 代码：激活函数可视化

```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-5, 5, 100)

# Sigmoid
sigmoid = 1 / (1 + np.exp(-x))
# Tanh
tanh = np.tanh(x)
# ReLU
relu = np.maximum(0, x)

plt.figure(figsize=(10, 4))
plt.subplot(1, 3, 1)
plt.plot(x, sigmoid)
plt.title('Sigmoid')
plt.axhline(0, color='gray', linewidth=0.5)

plt.subplot(1, 3, 2)
plt.plot(x, tanh)
plt.title('Tanh')
plt.axhline(0, color='gray', linewidth=0.5)

plt.subplot(1, 3, 3)
plt.plot(x, relu)
plt.title('ReLU')
plt.axhline(0, color='gray', linewidth=0.5)

plt.tight_layout()
plt.show()
```

### 💡 工程建议

- **隐藏层**：优先用 **ReLU**，简单高效
- **输出层做二分类**：用 **Sigmoid**（输出概率）
- **输出层做多分类**：用 **Softmax**（输出概率分布）
- **遇到 ReLU 神经元死亡**：换成 **Leaky ReLU** 或 **ELU**

## 5.4 反向传播（Backpropagation）

### 核心问题

网络有很多参数（权重 $W$ 和偏置 $b$），如何找到最优值？

**思路**：定义一个**损失函数**衡量预测与真实的差距，然后用**梯度下降**更新参数，让损失越来越小。

### 损失函数

| 任务类型 | 损失函数 | 公式 |
|---------|---------|------|
| 回归 | 均方误差（MSE）| $L = \frac{1}{n} \sum (y_i - \hat{y}_i)^2$ |
| 二分类 | 二元交叉熵（BCE）| $L = -[y \log(\hat{y}) + (1-y)\log(1-\hat{y})]$ |
| 多分类 | 交叉熵（CE）| $L = -\sum y_i \log(\hat{y}_i)$ |

### 梯度下降与反向传播

**梯度下降**：沿着损失函数梯度的反方向更新参数

$$
\theta_{new} = \theta_{old} - \eta \cdot \frac{\partial L}{\partial \theta}
$$

其中 $\eta$ 是**学习率**（Learning Rate）。

**反向传播**：用**链式法则**从输出层向输入层逐层计算梯度。

```
前向传播：输入 → 隐藏层 → 输出 → 计算损失
反向传播：损失 → 输出层梯度 → 隐藏层梯度 → 更新参数
```

### 代码：训练循环模板

```python
import torch.optim as optim

# 假设已有：model, train_loader, criterion, device

optimizer = optim.Adam(model.parameters(), lr=0.001)
criterion = nn.CrossEntropyLoss()

for epoch in range(num_epochs):
    for batch_x, batch_y in train_loader:
        # 1. 前向传播
        outputs = model(batch_x)
        loss = criterion(outputs, batch_y)
        
        # 2. 反向传播
        optimizer.zero_grad()   # 清空旧梯度
        loss.backward()          # 计算新梯度
        optimizer.step()         # 更新参数
    
    print(f"Epoch [{epoch+1}/{num_epochs}], Loss: {loss.item():.4f}")
```

### ⚠️ 注意：梯度消失与梯度爆炸

| 问题 | 现象 | 原因 | 解决方案 |
|------|------|------|---------|
| **梯度消失** | 前面层的梯度趋近于 0 | Sigmoid/Tanh 在两端梯度接近 0 | 用 ReLU、残差连接、更好的初始化 |
| **梯度爆炸** | 梯度变得极大 | 连乘效应导致梯度指数增长 | 梯度裁剪（Gradient Clipping） |

## 5.5 用神经网络做文本分类

### 从 TF-IDF 到神经网络

第 2~4 章我们用传统方法做文本分类（朴素贝叶斯、LSA），现在用神经网络：

```
文本 → 分词 → TF-IDF 向量 → 神经网络 → 分类结果
                (10000 维)      (隐藏层)    (正面/负面)
```

### 完整代码示例

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.model_selection import train_test_split
import torch
from torch.utils.data import DataLoader, TensorDataset

# ---------- 1. 数据准备 ----------
docs = ["I love this product", "Terrible experience", "Amazing quality", "Waste of money"]
labels = [1, 0, 1, 0]  # 1=正面, 0=负面

# TF-IDF 向量化
vectorizer = TfidfVectorizer(max_features=1000)
X = vectorizer.fit_transform(docs).toarray()
y = np.array(labels)

# 划分训练集/测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 转成 PyTorch 张量
train_dataset = TensorDataset(torch.FloatTensor(X_train), torch.LongTensor(y_train))
train_loader = DataLoader(train_dataset, batch_size=2, shuffle=True)

# ---------- 2. 定义模型 ----------
class TextClassifier(nn.Module):
    def __init__(self, input_dim, hidden_dim, num_classes):
        super().__init__()
        self.fc1 = nn.Linear(input_dim, hidden_dim)
        self.relu = nn.ReLU()
        self.dropout = nn.Dropout(0.3)  # 防止过拟合
        self.fc2 = nn.Linear(hidden_dim, num_classes)
    
    def forward(self, x):
        x = self.relu(self.fc1(x))
        x = self.dropout(x)
        x = self.fc2(x)
        return x

model = TextClassifier(input_dim=1000, hidden_dim=128, num_classes=2)

# ---------- 3. 训练 ----------
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

for epoch in range(10):
    for batch_x, batch_y in train_loader:
        outputs = model(batch_x)
        loss = criterion(outputs, batch_y)
        
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
    
    print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")

# ---------- 4. 预测 ----------
model.eval()
with torch.no_grad():
    test_output = model(torch.FloatTensor(X_test))
    predicted = torch.argmax(test_output, dim=1)
    print(f"预测结果: {predicted.numpy()}")
```

### 神经网络 vs 传统方法

| 维度 | 朴素贝叶斯 | LSA | 神经网络 |
|------|-----------|-----|---------|
| **特征处理** | 词袋/TF-IDF | TF-IDF + SVD 降维 | TF-IDF 或词嵌入 |
| **非线性** | ❌ 无 | ❌ 无 | ✅ 有 |
| **特征学习** | ❌ 人工设计 | ❌ 线性组合 | ✅ 自动学习非线性组合 |
| **训练速度** | 极快 | 快 | 较慢 |
| **可解释性** | 强（词概率） | 中等 | 弱（黑盒） |
| **数据需求** | 少 | 少 | 多 |
| **典型准确率** | ~85% | ~87% | ~90%+ |

## 关键概念速查

### 5.1 ~ 5.2 网络结构

| 术语 | 英文 | 含义 |
|------|------|------|
| 感知机 | Perceptron | 最简单的线性二分类器 |
| 前馈神经网络 | Feedforward Neural Network (FFNN) | 信息单向传播的多层网络 |
| 多层感知机 | Multi-Layer Perceptron (MLP) | 含隐藏层的 FFNN |
| 输入层 | Input Layer | 接收原始特征的层 |
| 隐藏层 | Hidden Layer | 学习特征组合的中间层 |
| 输出层 | Output Layer | 产生预测结果的层 |
| 前向传播 | Forward Propagation | 从输入到输出的计算过程 |
| 权重 | Weight | 连接神经元的参数，控制信号强度 |
| 偏置 | Bias | 神经元的阈值偏移量 |

### 5.3 激活函数

| 术语 | 英文 | 含义 |
|------|------|------|
| 激活函数 | Activation Function | 引入非线性的变换函数 |
| Sigmoid | Sigmoid | 将输入压缩到 (0, 1) 的 S 型函数 |
| Tanh | Hyperbolic Tangent | 将输入压缩到 (-1, 1) 的 S 型函数 |
| ReLU | Rectified Linear Unit | 负值截断为 0，正值保持不变 |
| Softmax | Softmax | 将向量转为概率分布（多分类输出） |

### 5.4 训练

| 术语 | 英文 | 含义 |
|------|------|------|
| 损失函数 | Loss Function | 衡量预测与真实值差距的函数 |
| 交叉熵 | Cross-Entropy | 分类任务常用的损失函数 |
| 梯度下降 | Gradient Descent | 沿梯度反方向更新参数的优化算法 |
| 反向传播 | Backpropagation | 用链式法则计算各层梯度的算法 |
| 学习率 | Learning Rate | 每次参数更新的步长 |
| 梯度消失 | Vanishing Gradient | 深层网络中梯度逐层衰减的问题 |
| 梯度爆炸 | Exploding Gradient | 深层网络中梯度逐层放大的问题 |
| Dropout | Dropout | 随机丢弃神经元，防止过拟合 |
| Adam | Adam Optimizer | 自适应学习率的优化算法 |
