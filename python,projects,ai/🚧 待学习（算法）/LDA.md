## 待学习清单

### 如果学的是 Linear Discriminant Analysis
- [ ] 核心思想：最大化类间散度 / 最小化类内散度
- [ ] 数学推导：广义特征值问题 $S_b w = \lambda S_w w$
- [ ] 与 PCA 的区别（有监督 vs 无监督）
- [ ] 多分类 LDA 的扩展（投影到 K-1 维）
- [ ] 代码实现：`sklearn.discriminant_analysis.LinearDiscriminantAnalysis`

### 如果学的是 Latent Dirichlet Allocation
- [ ] 核心思想：文档 = 主题分布，主题 = 词分布
- [ ] 数学基础：狄利克雷分布、贝叶斯推断、吉布斯采样
- [ ] 与 LSA 的区别（概率生成模型 vs 矩阵分解）
- [ ] 主题可解释性：如何理解 $\alpha$ 和 $\beta$ 超参数
- [ ] 代码实现：`sklearn.decomposition.LatentDirichletAllocation`
