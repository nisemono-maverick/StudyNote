#### ndarray 基本运算
- 大小相等的数组之间的任何算术运算都会将运算应用到元素级。
- 对于维度不同的数组间的运行，numpy 会启用广播机制将维度对齐。
![[Pasted image 20260611223238.png]]
- 直接通过 * 运算符和 `np.multiply()` 队两个数组进行相乘是对应元素乘法而非矩阵乘法
- 而使用 `np.dot()、ndarray.dot()、@` 可以进行矩阵乘法运算

#### 关于 axis 参数

NumPy 的统计函数默认会对数组的**所有元素**进行计算，返回一个标量。如果想按某个维度分别计算，就需要使用 `axis` 参数。

**核心规则**：`axis` 指定的是"要被折叠掉的维度"。

- `axis=0`：沿第 0 轴（行方向）折叠，即**按列计算**，结果反映每一列的统计量。
- `axis=1`：沿第 1 轴（列方向）折叠，即**按行计算**，结果反映每一行的统计量。
- 不指定 `axis`：对所有元素计算，返回一个标量。

```python
arr = np.array([[1, 2, 3],
                [4, 5, 6]])

# 不指定 axis：全局求和
np.sum(arr)              # 21

# axis=0：按列求和，结果形状为 (3,)
np.sum(arr, axis=0)      # array([5, 7, 9])

# axis=1：按行求和，结果形状为 (2,)
np.sum(arr, axis=1)      # array([ 6, 15])
```

**形状变化规律**：原数组 `shape=(2, 3)`，沿 `axis=0` 计算后消去第 0 维，结果 `shape=(3,)`；沿 `axis=1` 计算后消去第 1 维，结果 `shape=(2,)`。也就是说，结果形状 = 原形状去掉 `axis` 对应的维度。

**高维数组同理**：对于三维数组 `shape=(2, 3, 4)`，`np.sum(arr, axis=0)` 的结果形状为 `(3, 4)`；`np.sum(arr, axis=1)` 的结果形状为 `(2, 4)`。

**同时指定多个轴**：可以用元组一次性沿多个维度折叠。

```python
# 沿前两个轴求和，保留最后一个轴
np.sum(arr_3d, axis=(0, 1))   # shape 为 (4,)
```

**记忆口诀**：
> `axis=0` 竖着压（按列算），`axis=1` 横着压（按行算）；被压扁的维度消失，剩下的维度组成结果形状。

| `np.logical_and(a, b)` | 逻辑与（逐元素） | `np.logical_and([True, False], [True, True])` | `[True, False]` |
| `np.logical_or(a, b)` | 逻辑或（逐元素） | `np.logical_or([True, False], [False, False])` | `[True, False]` |
| `np.logical_not(a)` | 逻辑非（逐元素） | `np.logical_not([True, False])` | `[False, True]` |
| `np.any(a)` | 检查数组中是否至少有一个元素为 True | `np.any([0, 1, 0])` | `True` |
| `np.all(a)` | 检查数组中是否所有元素都为 True | `np.all([1, 2, 0])` | `False` |
| `np.where(condition, x, y)` | 根据条件选择元素 | `np.where([True, False, True], 1, 0)` | `[1, 0, 1]` |

> **比较运算符与函数对应**：`>` 等价于 `np.greater()`，`<` 等价于 `np.less()`，`==` 等价于 `np.equal()`，`!=` 等价于 `np.not_equal()`，`>=` 等价于 `np.greater_equal()`，`<=` 等价于 `np.less_equal()`。直接写运算符更简洁，但函数形式在需要作为参数传递时更有用。

#### np.where 详解

`np.where(condition, x, y)` 是条件选择函数，满足 `condition` 的位置取 `x`，否则取 `y`。`x` 和 `y` 可以是数组或标量。

```python
arr = np.array([1, 2, 3, 4, 5])

# 大于 3 的保留原值，否则替换为 0
np.where(arr > 3, arr, 0)
# array([0, 0, 0, 4, 5])

# 只传一个参数时返回满足条件的索引
np.where(arr > 3)
# (array([3, 4]),)
```

常见用途：

- 缺失值填充：`np.where(np.isnan(arr), 0, arr)`
- 数据分段：`np.where(arr >= 90, "A", np.where(arr >= 60, "B", "C"))`
- 异常值截断：`np.where(arr > 100, 100, arr)`
- 生成掩码：`np.where(arr > 0, 1, 0)`

### Series
#### 什么是 Series
类似于 NumPy 一维数组，但增加了 "标签"，可以理解为「一维标签化数组」
Series 是 Pandas 中的一个核心数据结构，类似于一个一维的数组，具有数据和索引。
Series 可以存储任何数据类型（整数、浮点数、字符串等），并通过标签（索引）来访问元素。Series 的数据结构是非常有用的，因为它可以处理各种数据类型。

#### 创建 Series
- 直接通过列表创建 
```python
Python  
import pandas as pd  
s = pd.Series([4, 7, -5, 3])  
print(s)  
# 0    4  
# 1    7  
# 2   -5  
# 3    3  
# dtype: int64
```
- 通过列表创建时指定索引
```python
s = pd.Series([4, 7, -5, 3], index=["a", "b", "c", "d"])  
print(s)  
# a    4  
# b    7  
# c   -5  
# d    3  
# dtype: int64
```
- 通过列表创建Series时指定索引和名称 `s = pd.Series([4, 7, -5, 3], index=["a", "b", "c", "d"],name="hello_python")`
- 通过字典创建Series
```python
dic = {"a": 4, "b": 7, "c": -5, "d": 3}  
s = pd.Series(dic)  
print(s)  
# a    4  
# b    7  
# c   -5  
# d    3
# dtype: int64  
s1 = pd.Series(dic,index=["a","c"],name="aacc")  
print(s1)  
# a    4  
# c   -5  
# Name: aacc, dtype: int64
```
