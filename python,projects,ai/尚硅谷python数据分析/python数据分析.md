## 1 numpy
### 1.1 ndarray
#### ndarray 的核心特性
- 多维性：高维数组
- 同质性：数组元素类型一致（通过 `dtype` 指定）
- 高效性
#### ndarray 的属性
- 以 `arr = np.array ([[1, 2], [3, 4]])` 为例
- 常用属性如下：

| 属性名称  | 通俗解释      | 使用示例      | 输出结果           |
| ----- | --------- | --------- | -------------- |
| shape | **数组的形状** | arr.shape | (2, 2)         |
| ndim  | **维度数量**  | arr.ndim  | 2              |
| size  | **总元素个数** | arr.size  | 4              |
| dtype | **元素类型**  | arr.dtype | int64（或 int32） |

- 其它属性如下：

| 属性名称     | 通俗解释                          | 示例代码         | 输出结果                  |
| -------- | ----------------------------- | ------------ | --------------------- |
| T        | **转置**                        | arr.T        | `[[1, 3], [2, 4]]`    |
| itemsize | **单个元素占用的内存字节数**              | arr.itemsize | 8（int64 类型占8字节）       |
| nbytes   | **数组总内存占用量**：size * itemsize。 | arr.nbytes   | 32（4元素 × 8字节）         |
| flags    | **内存存储方式**：是否连续存储（高级优化）。      | arr.flags    | C_CONTIGUOUS : True 等 |

#### ndarray 基本运算
- 大小相等的数组之间的任何算术运算都会将运算应用到元素级。
- 对于维度不同的数组间的运行，numpy 会启用广播机制将维度对齐。
![[Pasted image 20260611223238.png]]
- 直接通过 * 运算符和 `np.multiply()` 队两个数组进行相乘是对应元素乘法而非矩阵乘法
- 而使用 `np.dot()、ndarray.dot()、@` 可以进行矩阵乘法运算
