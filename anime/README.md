## 研究方法论

### 看什么
1. **叙事结构**：如何讲故事（POV、时间线、伏笔）
2. **思想内核**：作者想表达什么（反战、存在主义、后现代）
3. **历史语境**：创作时的社会背景（泡沫经济、冷战、互联网时代）
4. **互文性**：与其他作品的对话（致敬、解构、颠覆）

### 怎么看
1. 先看作品本身，记录直观感受
2. 查作者访谈和创作笔记
3. 读学术分析（东亚/日本研究、文化研究、媒体研究）
4. 对比不同解读，形成自己的观点

### 笔记格式
- 观影/阅读记录
- 核心观点提炼
- 待研究问题
- 资源清单
- 行动计划
